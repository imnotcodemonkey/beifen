同等学力英语高频词汇汇总35天速记 DAY15

时间:2010-07-26 来源:未知 点击:次 Tag: 同等学力 英语
同等学力英语高频词汇汇总35天速记 DAY15

同等学力英语高频词汇汇总35天速记 DAY FIFTEEN

basic 基本的，初级的

basic a. 基本的，初级的

basic need 基本需求

basis n. 基础，根据

on the basis of 根据…，以…为基础

base v. 以…作根据， 基于

base on 以…(为)根据

basically ad. 基本上，主要的

*fundamental a. 基本的，根本的 n. 基本原则

fundamental human rights 基本人权//

fundamental interests 根本利益

*essential a. 必要的；本质的，基本的

necessary a. 必要的，必需的

*primary a. 最初的，初级的；首要的，基本的；原

始的

primary forests 原始森林 // a primary school 小

学

primitive a. 原始的；简单的 n.原（始）人，

或事物

underlie  vt.位于……之下，成为……的基础

*junior n.下级，晚辈，大学三年级学生 a. 年少

的，下级的

*senior a. 高级的；较高的

elementary a. 初等的；基本的

because 因为

because conj. 因为

as conj. 因为 ad. ＆ prep. 同样地，作为

as if 好像；仿佛 // as long as 只要 // as soon as...

一…就… // as well 也，同样地

for conj. 因为 prep. 往，向，达；为了；长达...

for a while 暂时；一时 // for ever 永远

for about a month 长达一个月

*since prep.& conj. 由于，自从 ad. 从那时以来

since then 从那时起 // ever since 从那以后(一直)

because 引导的从句表示的是直接而不是推断的原因，

语气更强，置于主句前或后皆可，回答why 提出的问题时

只能用because，并且不能与so 连用；

as 引导的从句置于主句前；

for 引导的从句置于主句后；

since 含有“既然”之意，引导的从句置于主句前。

so conj. 因此，所以 pron.& ad. 那(这)么，那样(表

示程度)

so far 到目前为止 // so...that 如此…以至于

cause vt. 引起，使发生

cause trouble to sb. 给某人造成麻烦// cause sth / sb.

to do sth 使某物或某人做某事

*excuse n. 借口，辩解，原谅

excuse oneself 替自己辩解，婉言谢绝

without excuse 无故地，故意

sake n. 缘故，理由

reason n.原因，理由 vt. 讲道理，推理

by reason of 因为 //

for this reason 为此，出于这种原因

cause 指造成某种后果的原因，常说the cause of sth。

Reason 指产生某种行为的理由，常说the reason for

sth。

Excuse 借口，编造的假理由。

*owe v. 欠债; 归功于，感激

owing to 由于

therefore ad. 因此，于是

thus ad. 如此，这样；这么，因此
