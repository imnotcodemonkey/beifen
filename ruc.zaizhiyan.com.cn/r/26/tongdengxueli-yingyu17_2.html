康状况不佳 //

on condition that 条件是，如果

unconditional a. 无条件的，绝对的，无限制的

case] n.箱子，盒子；情形，状况； 实例，案例

in case 以防 // in case of 万一发生… // in any case

无论如何 //

in no case 绝不，在任何情况下都不 // in this / that

case 假若这样/那样的话

state n. 状态，情况，情形(常用单数)；国家， vt.

陈述，说明

in a / an ... state 处于…状态

anarchy n. 无政府；混乱状态

*emergency n. 紧急情况

cause / create an emergency 引发紧急状况

declare a state of emergency 宣布紧急状况

*crisis n. 危急关头，（政治、经济上的）危机，决定

性的时刻

situation n.位置，地点；形势，境况，局面

background n. 背景

circumstance n. 环境，条件，形势

under (in) the circumstances 在此情况下，情况既然

如此 //

under (in) no circumstances 无论如何不，决不

*phenomenon n. 现象，迹象

deadlock n. 僵局

connect 连接，联系

*connect vt. 连接，相连，联系

connect to 连接到… // connect with 和…连接起来

*connection n. 联系

in this / that connection 由于这/那事，为此 //

in connection with 与…有关或相关

join vt.&a. vt. 连接，结合，参加

join in 加入 // join together 连在一起

tie [tai]vt. (用绳，线)系，拴，扎

tie...to... 把…系在…上

*stick vt. 粘贴 n. 木棒

stick to 同…粘在一起

bond n. 纽带，联系 vt.&a.（使）黏合，（使）结合

bundle n. 一包，一捆，一束 vt.&a. 收集，归拢

fasten vt. 扎牢，扣牢

fasten one’s attention/eyes on sth 将注意力/目光

集中在某事上 //

fasten up 关紧，捆牢

lace n. 网眼花边，花边； 鞋带，系带 vt. (up)用

系带束紧

weave vt. 织，编

knit vt. 编结，编织；使紧密地结合，使紧凑；皱紧，

皱（眉） vi.编结，编织； (骨折) 愈合

*unity n. 统一，整体；一致，团结，协调

union n. 联合，联合会

unite v. 合并，联合；(使)团结，结合，混合

united a. 联合的，团结的，统一的

the United Nations 联合国 // the United States 美

利坚合众国

*associate vi. 结交，交往 vt. 使联合，使有联

系 n. 伙伴

associate ... with （思想上）联系，（业务上）合伙；

//

associate with 与…交往

*socialize vt. 使合群；使适应社会的需要

*contact n.& vt. 联系，接触，交往

touch vt. 接触, 触摸

association n. 协会，社团; 联合，结合，交往

attach vt. 系，贴，连接；使附属

be attached to 附属于… // attach oneself to 加入；

接近

*concern n.关联； 关心 vt. 涉及

as /so far as...be concern 就…而言 //

concern oneself about 关心，挂念 //

have no concern with 同…无关 //

show concern for sb. 关心某人

*involve vt. 卷入，陷入；包含，含有

relevant a. 紧密相关的；切题的

be relevant to 与…密切相关

combine vt. 使联合；使结合

connected a. 相连的，接通的

division n. 分，划分

link vt. 连接，联系

mix vt. 混合

mix up 拌和，搅匀，混淆 // mix ... with ... 把…

和…混和

mixture n. 混合物

melt n. 熔化,熔化物 vt. 熔化,溶解,渐混

mingle vt. 使混合，使相混 vi.混合起来, 相混合; 相

交往，相往来

weld v. 焊接

loose a. 宽松的，不牢固的

get / break loose 逃出，挣脱

tight a. 紧身的，牢固的 ad. 紧地，满满地

separate vt. 分开的，分离的 a. 个别的

separate from 分隔开

*divide vt. 分开的，分离的

divide into 分割成

divide

divide ... into ... 指把某个整体分割为若干部分。

separate

separate ... from ... .表示 "将…与…分开"，指把原

来连在一起或靠近的分隔开来。

division n. 分开，分隔

cut vt.切割，剪; 削减，减少; 切断

cut down 砍倒 // cut in 插队 // cut off 割 //

cut out 删掉 // cut up 切碎

slice vt. 切，割，划 n. 薄片，切片

segment n. 片，部分，断片，段

smash vt.&n. 打碎，粉碎

shatter vt. 使粉碎，砸碎; 使破灭，vi. 碎裂

separate vt. 分开, 隔离; adj.分开的, 分离的, 个

别的, 单独的

separately ad. 分离地; 个别地, 单独地

separation n. 分离，隔离

isolate vt. 隔离，孤立

scatter vt. 撒，驱散，散开；散布 vi. 分散，消散

split vt. 劈开，裂开 a.分裂的 n. 裂缝，分化，

分裂

*diffuse v.扩散,使弥漫

*disperse vi.散开, 分散 vt .使消散,驱散

apart ad. 相距；分开

apart from 除去

undo vt. 解开，松开

uncover vt. 揭开, 揭露

unload vi. 卸货, vt. 摆脱…之负担, 倾销, 卸

compartmentalize v. 把…分成各自独立的几部分

*consider vt .&vi. 考虑;认为

consider ... as(to be)... 把…看作… //

consider doing sth 考虑做某事 //

considerate a. 体贴的，体谅的

be considerate of / to / toward sb. 对某人体贴

*considerable a. 值得考虑的；相当（大、多）的

consideration n. 考虑，关心

take ... into consideration 考虑… //

be under consideration 正在考虑中 // in

consideration of 考虑到…

*cherish vt. 珍爱, 珍惜；爱护；抱有

*thoughtful a. 深思的，沉思的；体贴的，关心的

brood vi. (on, over, about)沉思, 孵蛋n. (雏鸡等

的)一窝；(一个家庭的)所有孩子

*care vi. 关心，照顾

take care 留心，当心 // take care of 照顾，照料 //

care about 在乎，关心，喜欢 // care for 喜欢，想要 //
